Welcome to my website! This is my first dynamic web application. The purpose of this site is mainly to practice my Java and Spring skills in a live setting. I am hosting live on Amazon web services for free.

To access the contents of this website, which I assure you are incredibly entertaining to say the least, you will have to create a username and password.

This is a simple process that only takes seconds and requires only a choice of username and password, no personal information is required.

The username and password are enabled to showcase the abilities of the Spring Security framework for Java. Every username and password is stored in a database for use when you return to check out all the awesome goat pictures.
