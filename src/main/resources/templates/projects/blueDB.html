This is an application for managing a MySQL database, or many. Right now it is in its first stages. Still it is quite powerful. It allows the user to login to any host and view any database or table as long as the credentials are correct.

Cycle through entries to view the current status of the table and execute SQL statements as needed. Then simply click refresh and it will reload the table to see the executed statements effect.

Also the user has the option of creating new databases and tables or deleteing them. Simply enter a name for a new database and click create, or enter a SQL statement for the table and click create. Currently you can also drop tables through the create table button.

This is my database managment system. I use this program to edit and maintain the databases I use in test projects. The first image is of the main user interface where you can choose to either view and edit any table in any database at any host (as long as you know the credentials) or choose to make a brand new database or table.

On the view interface, a user must first log in to the database they want to edit. Next they are free to click through each entry and execute SQL statements as needed.

Cycle through the database viewing entries.

Execute SQL statements to affect the tables and then click the refresh button to see the changed results.

The results of the executed statement are now visible. Also users are able to create new databases and tables at will. Simply goto the create interface and enter the name of what you what in the field or a SQL statement for tables. Deleting a database is also avaiable on the view.
